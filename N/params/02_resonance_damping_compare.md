# Task 2 — resonance_damping_compare

## 物理
共振时阻尼系数决定稳态振幅。
   X_ss = (m·A) / (c·ω₀)

三个相同 k=100/M=0.1 的振子，仅 joint damping c 不同 — 同步驱动在 ω₀。

## 几何
- 三个并排弹簧-质量振子，x 间距 0.3 m（x = −0.30, 0, +0.30）
- 共用木质支架（横梁宽 1.30 m）
- 质量块：0.06×0.06×0.06 m 立方体

## 物理参数
| 振子 | 颜色 | joint damping c | Q ≈ ω₀·M/c | 预测稳态振幅 |
|------|------|----------------|------------|-------------|
| A | 红 | 0.01 | 316（高 Q）| 222 mm |
| B | 琥珀 | 0.63 | 5.0 | 3.5 mm |
| C | 蓝 | 2.0 | 1.58（低 Q）| 1.1 mm |

| 共同参数 | 值 |
|---------|---|
| 弹簧刚度 k | 100 N/m |
| 质量 M | 0.1 kg |
| 自然频率 ω₀ | 31.62 rad/s |
| 驱动加速度 A | 0.7 m/s² |
| 驱动频率 | ω₀（同步共振） |

注：高 Q 振子建立时间 τ = 2m/c = 20 s，所以在 20 s 渲染中只达到 ~63%
的稳态振幅（约 140 mm）。

## 驱动
`gen_resonance_damping_compare.py` 对所有三个 slide joint 注入：
```
data.qvel[slide_X] += A * sin(ω₀ * t) * dt
```

## 仿真
- timestep 0.001 s
- 渲染 20 s（让高 Q 充分起振，约 τ 个时间常数）

## 实测峰值
- A（低阻尼）：144 mm
- B（中阻尼）：16 mm
- C（高阻尼）：12 mm
比率约 9 : 1 : 0.75
