# Task 11 — brazil_nut_granular_effect

## 物理
颗粒分离的"巴西果效应"：
垂直振动颗粒混合物时，大颗粒会上升到顶部。
机制：
1. 大颗粒下方的小颗粒在垂直振动时填入空隙（void-filling）
2. 振动诱导的对流胞将大颗粒"挤"到表面
两个机制都让大颗粒在每个振动周期里净向上移动。

## 几何
- 圆柱容器：R = 0.065 m，H = 0.18 m
- 壁：24 块薄 box geom 环成圆筒（4 mm 厚）+ 半透明材质（alpha 0.30）
  让侧视摄像机能看到内部
- 底部：圆柱（厚 14 mm，重 4 kg）
- 容器作为整体由 slide z joint 连到 world，被 position actuator 驱动

球：
- 1 个大红球：R = 0.020 m，M = 0.07 kg
- 50 个小奶油色球：R = 0.007 m，M = 0.003 kg

## 物理参数
| 参数 | 值 |
|------|---|
| 容器 R | 0.065 m |
| 容器 H | 0.18 m |
| 容器质量 | 4 kg |
| 振动振幅 A | 0.012 m |
| 振动频率 f | 8 Hz |
| 振动角频率 ω | 2π·f ≈ 50.3 rad/s |
| 大球摩擦 | 0.40 |
| 小球摩擦 | 0.40 |

## 初始位置
- 大红球：(0, 0, 0.10 + 0.014 + 0.020 + 0.001) = (0, 0, 0.135)（沉在底部中央）
- 50 个小球：堆在大球上方，5-6 层（hex pack），从 z=0.144 到 0.215
- 容器中心在 world (0, 0, 0.30)

## 驱动方式
Position actuator 驱动容器 z：
```
<position name="vib_drive" joint="container_z" kp="20000" kv="80"
          ctrlrange="-0.03 0.03"/>
```
gen 脚本：
```
data.ctrl[vib_drive] = AMP_VIB · sin(2π·f·t)
```

## 仿真
- timestep 0.0005 s（小 dt 处理 8 Hz 振动 + 接触）
- impratio 4，cone elliptic
- 渲染 10 s

## 实测结果
- 大球初始 z = 30 mm（相对桶底）
- 大球终态 z = 93 mm
- 上升 63 mm，跨越 ~50% 桶高
- 巴西果效应清晰
