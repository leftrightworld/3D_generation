# Task 6 — rotating_bucket_parabola

## 物理
旋转参考系中，离心势 + 重力 = 有效"向外向下"的等效重力。
小球在旋转圆筒里堆成抛物面：
   z(r) = z₀ + ω²·r² / (2g)
预测的下凹深度（rim 比中心高）：
   Δh = ω²·R² / (2g)

## 几何
- 圆筒：R = 0.12 m，H = 0.18 m
- 壁：36 块薄 box geom 环成圆筒（厚 4 mm）
- 底部：圆柱（厚 14 mm，重 18 kg 防止反扭矩）
- 圆筒置于 4cm 高木质底座上
- 渲染 60 个小球：3 红 / 3 琥珀 / 3 蓝（交替着色便于追踪）

## 物理参数
| 参数 | 值 |
|------|---|
| 圆筒内半径 R | 0.12 m |
| 圆筒高 H | 0.18 m |
| 圆筒质量 | 18 kg |
| 圆筒角速度 ω | 8 rad/s |
| 小球半径 R_ball | 0.006 m |
| 小球质量 M_ball | 0.002 kg |
| 小球数 N | 60（5 层 × 12 球）|
| 球-球摩擦 | 0.3 |
| 球-壁摩擦 | 0.3 |

## 预测
Δh = 64 · 0.0144 / (2 · 9.81) ≈ 47 mm
（在 R=0.12 边缘比中心高 47mm，跨越约 4 个球直径）

## 初始条件
- 圆筒：qvel(bucket_spin) = 8 rad/s
- 60 个小球紧密堆在中央柱状区域（r < 0.025 m）：
  - 5 层 × 每层 12 球（1 中心 + 6 内环 r=0.013 + 5 外环 r=0.024）
  - z 从 0.118 到 0.174 m（世界坐标）
- 每个小球初速 v = (−ω·y, +ω·x, 0)（与所在位置同步共转）
- 每个小球 spin wz = ω（避免与桶之间的滚动滑动）

## 仿真
- timestep 0.001 s
- impratio 4，cone elliptic（处理大量接触）
- 渲染 8.0 s（球散开 → 撞墙 → 沉淀到抛物面）

## 故事感
1. t=0：球在中央堆叠
2. t≈1-2 s：球被甩到墙边
3. t≈3-4 s：开始形成边缘环 + 中央空心
4. t≈5-8 s：稳定的抛物面分布

## 相机
3/4 overhead，pos (0.30, −0.30, 0.65), fovy 50
xyaxes "0.707 0.707 0  −0.500 0.500 0.707"
（从右上 ~60° elevation 俯视，看清桶内）
