# Task 3 — wave_dispersion_chain

## 物理
弦上横波的色散关系：
- 均匀弹簧链 → 脉冲传播形状不变（线性弦近似下）
- 交替弹簧链（k₁/k₂）→ 出现声学/光学分支 + 带隙，
  高频成分滞留 / 散射，脉冲展开为多个 lobes

## 几何
- 两条 20 节点链垂直堆叠（链 A 下，链 B 上）
- 节点间距 L_seg = 0.050 m，链长 0.95 m
- 节点球：R = 0.020 m
- 链 A 节点：蓝色（统一 k = 500）
- 链 B 节点：琥珀色（交替 k₁ = 200 / k₂ = 800）
- 端点（i=0 和 i=N−1）pinned（深灰色，无 slide joint）

## 物理参数
| 参数 | 值 |
|------|---|
| 节点数 N | 20 |
| 节点质量 m | 0.04 kg |
| 节点间距 L_seg | 0.050 m |
| Tendon 自然长 L_rest | 0.010 m（预拉伸 0.040 m → 提供张力线性化横波） |
| 链 A 弹簧 k_A | 500 N/m → 张力 T_A = 20 N |
| 链 B 弹簧 k_B | 交替 (200, 800) N/m |
| Tendon 阻尼 | 0 |

预拉伸的关键作用：无预拉伸时横向位移的最低阶恢复力是 O(δz³)，
波无法传播。加预拉伸后 leading 项是线性的 T·d²z/dx²。

## 初始条件
所有内部节点（i=1..18）的 z 位置设置为：
```
z_i = AMP * exp(−(i − I₀)² / σ²)
```
| 脉冲参数 | 值 |
|---------|---|
| 振幅 AMP | 0.025 m |
| 中心 I₀ | 4 |
| σ² | 4.0（适中宽度，更多低频成分） |

## 波速预测
c_A = √(T_A / μ) = √(20 / 0.8) = 5.0 m/s
单次穿越时间：0.95 / 5.0 = 0.19 s

## 仿真
- timestep 0.0005 s
- gravity = 0（不影响横向运动但保持纯净）
- 渲染 1.2 s（约 6 次穿越，前 1-2 次最清晰显示色散对比）

## 偏离 spec 之处
原 spec：N=40, m=0.01, 渲染 3 s。
原参数下脉冲在 3 s 内反弹 10-15 次，色散信号被噪声淹没。
我用 N=20、m 更大（慢波速）、短渲染时长，让首次穿越的色散对比可读。
