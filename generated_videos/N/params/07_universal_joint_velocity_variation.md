# Task 7 — universal_joint_velocity_variation

## 物理
Hooke / Cardan 万向节传输等角速度的变化：
   tan(φ_out) = tan(φ_in) · cos(θ)
当输入轴匀速 ω_in 时，输出角速度按以下振荡：
   ω_out / ω_in = cos(θ) / (cos²(φ_in) + cos²(θ)·sin²(φ_in))
振荡频率 = 2·ω_in（每转 2 次），峰谷之间相隔 1/4 转。

θ = 30° 时：
- 最小值 ω_out/ω_in = cos(30°) ≈ 0.866
- 最大值 ω_out/ω_in = 1/cos(30°) ≈ 1.155
（即 ±15.5% 的变化）

## 几何
- 两根轴在原点交汇，夹角 θ = 30°
- 输入轴沿 world +y 方向
- 输出轴在 yz 平面内绕 x 轴倾斜 30°
- 每根轴端有圆盘标记（蓝色 = 输入，红色 = 输出）+ 偏置奶油色 sector
- 静态可见的"+" 形十字件（黑色），仅装饰，不参与运动

## 物理参数
| 参数 | 值 |
|------|---|
| 输入角速度 ω_in | 4 rad/s |
| 轴间夹角 θ | 30°（π/6 rad）|
| ω_out/ω_in 理论范围 | 0.866 .. 1.155 |
| 实测范围 | 0.866 .. 1.155（完全吻合）|

## 驱动方式
**运动学驱动**（不是物理仿真闭合机构）。
真实 U-joint 需要 closed-loop kinematics，难以在 MuJoCo 直接搭建。
取而代之：gen 脚本每帧直接设置 qpos：
```
φ_in(t) = ω_in · t
φ_out(t) = atan2(sin(φ_in)·cos(θ), cos(φ_in)) + revs·π
```
其中 revs = floor(φ_in / π)（让角度连续不跳变）。

## 仿真
- timestep 0.001 s
- gravity = 0（不影响纯运动学）
- 渲染 4 s（约 2.5 转 = 5 个 ω_out 振荡周期）
