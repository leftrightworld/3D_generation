# Task 4 — melde_harmonic_3modes

## 物理
弦的本征模态（两端固定）：
   z_i^(n) = A·sin(n·π·i/(N−1))
n=1 / 2 / 3 分别对应一个/两个/三个 antinode（驻波形状）。

## 几何
- 三条 40 节点弦垂直堆叠
- 每条全长 L = 0.50 m，节点间距 L_seg = 0.50/39 ≈ 12.82 mm
- 节点球：R = 0.011 m
- 端点 pinned，内部节点 z-slide joint

z 高度：
| 弦 | 颜色 | 模式 | z_base |
|----|------|------|--------|
| 1 | 红 | n=1（1 个 antinode）| 0.30 |
| 2 | 琥珀 | n=2（2 个 antinode）| 0.50 |
| 3 | 蓝 | n=3（3 个 antinode）| 0.70 |

## 物理参数
| 参数 | 值 |
|------|---|
| 节点数 N | 40 |
| 节点质量 m | 0.005 kg |
| 节点间距 L_seg | 12.82 mm |
| Tendon 弹簧刚度 k | 200 N/m |
| Tendon 自然长 L_rest | 0.001 m（最大预拉伸 → T = 200·0.01182 = 2.36 N）|
| 阻尼 | 0 |
| 模态振幅 A | 0.008 m |

## 初始条件
对每条弦 prefix（"m1"/"m2"/"m3"）的内部节点 i=1..38：
```
data.qpos[z_<prefix>_i] = A * sin(n·π·i/(N−1))
```
qvel = 0 — 释放后弦从最大位移开始振荡。

## 预期频率
波速 c = √(T·L_seg/m) ≈ 2.46 m/s
- 模式 1：f₁ = c/(2L) ≈ 2.46 Hz，T₁ ≈ 0.406 s
- 模式 2：f₂ ≈ 4.92 Hz，T₂ ≈ 0.203 s
- 模式 3：f₃ ≈ 7.39 Hz，T₃ ≈ 0.135 s

## 仿真
- timestep 0.0002 s（小步长保证 mode 3 稳定）
- integrator RK4（更好的能量守恒）
- gravity = 0
- 渲染 1.5 s（mode 1 约 3.7 周期）

## 已知限制
即使在线性化区间，离散链上的非线性耦合（高阶项 ~ k·δz²/L_seg）
会在 ~1 周期内把能量分散到其它模态。实测：
- mode 1 在 t < 0.5 s 内仍清晰可辨
- 之后所有模态的可见振幅快速衰减到接近零
demo 主要展示初始模态形状的对比（frame 1 清晰）+ 前一个周期的振荡
