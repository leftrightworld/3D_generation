# Task 8 — snells_law_ball_refraction

## 物理（Snell 类比）
Snell 定律的力学类比：球在两个不同摩擦区间的边界处发生轨迹偏转。
- 沿边界的动量分量（parallel）守恒
- 垂直分量改变 → 角度从 θ₁ 变到 θ₂

实际 Snell 是：n₁·sin(θ₁) = n₂·sin(θ₂)
若进入"较慢介质"（更高摩擦/折射率）→ 角度向 normal 偏。

## 几何
- 平面分两个半区：
  - x < 0：浅蓝色（zone_fast，无摩擦）
  - x > 0：琥珀色（zone_slow，摩擦 0.30）
- 边界处一条深色细线标记 x=0
- Puck：圆柱（避免滚动），R = 0.020，厚 0.010

## 物理参数
| 参数 | 值 |
|------|---|
| Puck 半径 | 0.020 m |
| Puck 质量 | 0.05 kg |
| Puck 摩擦设置 | "0 0 0"（与地板的有效摩擦由地板决定）|
| 快速区摩擦 | 0 |
| 慢速区摩擦 | 0.30 |
| 初始位置 | (−0.32, −0.10, 0.018) |
| 初始速度 | (1.5, 0.8, 0) m/s |
| 入射角（相对边界 normal）| arctan(0.8/1.5) ≈ 28.1° |
| 边界处 v_x 缩放系数 | 0.45 |

## 驱动方式
gen 脚本监视 puck.x，当首次 x > 0 时施加 impulse：
```
data.qvel[vx] *= 0.45  # v_y 不变
```
出射角变为 arctan(0.8 / (1.5·0.45)) = arctan(1.185) ≈ 49.9°

慢速区内的真实摩擦继续减速 puck。

## 仿真
- timestep 0.001 s
- 渲染 2.0 s，FPS 60（puck 移动快）

## 已知限制
- 真实摩擦方向与 v 反向，并非"垂直于边界"，所以无法靠真摩擦自然产生偏折
- 我用手动 impulse 实现 Snell-like 偏折
- **偏折方向是远离 normal**（28° → 50°），与 wave-optics Snell 的"进入更慢介质 → 向 normal 偏"方向相反。要向 normal 偏需要在边界处给 puck 加速，物理上无意义
- 视觉演示：边界处明显拐弯 + 进入黄色区后减速
